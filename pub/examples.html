TRPG.JS is a library that can be used to build web-based TRPGs.

What is TRPG?
Tabletop Role-Playing Game, where multiple players need to follow a set of rules, role-play their characters, and go through a story together.

You can use TRPG.JS library to display the information panel for each player, such as attributes, skills, talent tree and inventory, as well as to display events (and possible options/reactions to events) and build chat windows.

This library is mainly focusing on DOM manipulation, please be advised that you can modify the style of everything to make the UI prettier :)

// javascript constructor
const myBag = new Inventory('inventory', {
  quantity : { apple: 20,banana: 5,
               orange: 0,strawberry: 100},
  description: { apple: 'rotted',
           banana: 'it tastes like a banana' }, 
    image: {//path to image here},
    numRow: 4,
    numColumn: 6,
    menuOption: ['use','destroy'],
    menuOptionOnclick: { 'use': use_item,
     'destroy':  destroy_item,}  
});
Example 
                    
// left click menu option onclick functions
function use_item(item) {
  myBag.removeItem(item, 1);
}

function destroy_item(item) {
  myBag.clearItem(item);
}

function gift_item(item) {
  myBag.removeItem(item, 1);
  // print "thank you" above
}
