TRPG.JS

TRPG.JS is a library that can be used to build web-based TRPGs.

What is TRPG?
Tabletop Role-Playing Game, where multiple players need to follow a set of rules, role-play their characters, and go through a story together.

You can use TRPG.JS library to display the information panel for each player, such as attributes, skills, talent tree and inventory, as well as to display events and the possible options or reactions to the events.

Attribute Panel

A panel simply displays the attributesâ€™ names and values.
Try it on Play Ground

Getting Started

HTML:

<div id="attribute-panel"> </div>

JavaScript:

const aPanel = new AttributePanel(
    // first parameter: element ID
    'attribute-panel', {
    // second parameter: configurations
        // required configuration: attribute names and values
        attributes : {
            hp: 1000,
            mp: 500,
            sanity: 'healthy',
            career: 'web developer'
          },
        // optional configurations: description
        description : {
            hp: 'heath point, you will lose health when 
                you eat a rotten apple :)'
        },
        showDescription : true
  });

const aBar = new AttributeBars(

            hp: 'heath point, you will lose health when

Inventory

A table-like container for players to deal with their items.
Try it on Play Ground

Getting Started

HTML:

<div id="inventory"> </div>

JavaScript:

const myBag = new Inventory(
    // first parameter: element ID
    'inventory', {
    // second parameter: configurations
        // required configuration: item names and quantities
        quantity : { apple: 20,banana: 5,
                   orange: 0,strawberry: 100},
        // optional configurations: description
        description: { apple: 'rotted',
               banana: 'it tastes like a banana' }, 
        // optional configurations: image path
        image: { apple: //path to image here},
        // optional configurations: row number and column number
        numRow: 4,
        numColumn: 6,
        // optional configurations: left click menu options and their onclick event functions
        menuOption: ['use','destroy'],
        menuOptionOnclick: { 'use': use_item,
         'destroy':  destroy_item,}  
});

Event List

A window displays an event and its options.
Try it on Play Ground

Getting Started

HTML:

<div id="event-list"> </div>

JavaScript:

const eList = new EventList('event-list', {
  events : {  
    'golden apple': ['pick up','cancel'],
    'pikachu': ['fight', 'run'],
    'an event without description':
          ['an option without description','cancel']
  },
  eventDescription: {
    'golden apple': `There is a golden apple on the ground,
           and no creatures is guarding it nearby. When you 
           approach, you smell a scent that you can't resist.`,
    'pikachu': "A wild Pikachu appears!"
  },
  options: {
    'cancel': cancel,
    'run': run,
    'fight': fight
  },
  optionDescription: {
    'run': "I'm not strong enough, it's time to escape!",
    'fight': "Escape is for coward, I'd fight till death!",
    'pick up': "It's mine now! [Pick Up]"
  }
});

API Functions

These functions can be used directly.

Function Type Usage Example
show(event_name) event_name: String display a certain event show("golden apple")
showRandom() - display a random event showRandom()
showRandomWithoutRepeat() - display a random event without repeating the previous one showRandomWithoutRepeat()
setEvent(event_name, options, description) event_name: String, options: Array, description: String add or modify an event. Will display this event setEvent("pikachu", ["fight", "pet"], "You see a Pikachu in your bathroom")
setOption(option_name, click_function, description) option_name: String, click_function: function, description: String add or modify an option setOption("fight", fight, "I don't like it, so I will fight it")
deleteEvent(event_name) event_name: String delete an event. If currently displayed event is deleted, then will display the first event deleteEvent("golden apple")
addEventOption(event_name, option_name) event_name: String, option_name: String add an option to an event addEventOption("pikachu", "escape")
deleteEventOption(event_name, option_name) event_name: String, option_name: String delete an option from an event deleteEventOption("pikachu", "pet")
setOptionOnclick(option_name, click_function) option_name: String, click_function: function add or modify an optionâ€™s onclick function setOptionOnclick("escape", escape)
setOptionDescription(option_name, description) option_name:String, description: String add or modify an optionâ€™s description setOptionDescription("escape", "I'm too weak, I just want to run away")
